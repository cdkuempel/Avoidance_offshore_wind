This repository accompanies the publication "Identifying no-go areas in offshore wind planning is essential for Net Zero, Nature Positive future" by Caitlin D. Kuempel, Alexandra White, Stephen C.L. Watson and James E.M. Watson

Required data:

[Global Offshore Wind Infrastructure](https://doi.org/10.6084/m9.figshare.24309475 )

WDPA Marine protected area dataset (September 2025 version was used in the published analysis) is downloded in the 3rd Script below using the wdpar package.

To recreate the figures and results, run the scrips in numerical order.

The results of the policy review can be found in Supplementary Table 1 of the published text.
